# My CV Website
This project is a personal resume website for Msizi Msimango built using HTML, CSS, and JavaScript.
The webpage presents professional experience, education, skills, and contact information in a clean and responsive layout.

**Features**

- Responsive resume layout

- Smooth navigation scrolling

- Section reveal animations on scroll

- Professional profile header with contact links

- Structured sections for experience, education, skills, and references
